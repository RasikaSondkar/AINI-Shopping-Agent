AINI is a Python-based intelligent shopping assistant that compares product prices, ratings, and reviews across multiple platforms and recommends the best option.

## 🚀 Features

- 📊 Multi-platform price comparison (Amazon, Flipkart, Croma)
- ⭐ Rating & review analysis
- 🧠 Smart recommendation logic
- 💬 Interactive CLI chatbot
- 🔁 Compare multiple products
- 🗂 Modular project structure

## 🛠 Tech Stack

- Python 3
- Modular Architecture
- Rule-based AI logic

## 📦 Supported Products

- Laptop
- Smartwatch
- Headphones
